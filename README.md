#  About :rocket:



##  Installation :laptop:
Make sure your have Node.js installed on your computer.

Clone the repo:

git clone https://github.com/Gabriel2233/Discounter.git

After entering inside the project directory, start the project:

```bash
yarn
 or 
npm install
```

```bash
yarn dev
 or
npm run dev
```

That's it!
